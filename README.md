## 💻 Usage

The portfolio is ready to use out of the box. Simply open `portfolio.html` in any modern web browser. The website includes:

- **Navigation**: Use the navbar to jump between sections
- **Contact Form**: Fill out the contact form to send messages
- **Project Links**: Click on project cards to view GitHub repositories
- **Download Resume**: Use the PDF download button to get the resume

## 📱 Responsive Design

The portfolio is fully responsive and optimized for:

- **Desktop**: Full layout with side-by-side sections
- **Tablet**: Adjusted spacing and font sizes
- **Mobile**: Stacked layout with optimized touch targets
- **Small Screens**: Compact design with essential information

### Breakpoints:
- **Large screens**: 992px and above
- **Medium screens**: 768px - 991px
- **Small screens**: 480px - 767px
- **Extra small**: Below 480px

## 🔧 Customization

To customize this portfolio for your own use:

1. **Update Personal Information**:
   - Replace `arsal.jpeg` with your profile photo
   - Update name, title, and contact information in `portfolio.html`
   - Modify the about section with your details

2. **Add Your Projects**:
   - Update project cards in the projects section
   - Add your GitHub repository links
   - Include relevant technology badges

3. **Modify Styling**:
   - Edit `portfolioStyle.css` to change colors, fonts, or layout
   - Adjust hover effects and animations
   - Customize the gradient backgrounds

4. **Update Content**:
   - Modify work experience and education sections
   - Add or remove hobbies
   - Update the contact form action
